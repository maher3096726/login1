# 🎓 نظام تسجيل بيانات الطلاب - المعهد العالي للعلوم الإدارية ببلقاس

نظام ويب متكامل لتسجيل بيانات الطلاب واستخراج المعلومات تلقائياً من:
- 📄 **ورقة الاختيار** (اسم الطالب، رقم الجلوس، رقم الإيصال، رقم الهاتف)
- 🆔 **البطاقة الشخصية** (الرقم القومي)

## ✨ المميزات

- 📸 **استخراج تلقائي للبيانات** من الصور باستخدام EasyOCR
- 🎯 **دقة عالية** في التعرف على الأرقام العربية والإنجليزية
- 🌙 **وضع الظلام/النهار** لتجربة مريحة
- 📱 **تصميم متجاوب** يعمل على جميع الأجهزة (هواتف، أجهزة لوحية، كمبيوتر)
- 💾 **حفظ البيانات** في Firebase Realtime Database
- 📋 **أسئلة شائعة** تفاعلية

## 🛠️ التقنيات المستخدمة

| التقنية | الاستخدام |
|---------|-----------|
| **Flask** | إطار عمل الخادم الخلفي |
| **EasyOCR** | استخراج النص من الصور |
| **Pillow** | معالجة وتحسين الصور |
| **Firebase** | تخزين بيانات الطلاب |
| **HTML/CSS/JS** | واجهة المستخدم |
